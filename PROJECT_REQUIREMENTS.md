## PHẦN 13 — DANH SÁCH TÀI LIỆU / FILE CẦN GỬI KÈM

> Vui lòng gửi các file sau qua kênh đã thống nhất với đội lập trình (không gửi qua email thông thường để đảm bảo bảo mật).

### Tài liệu bắt buộc — thiếu thì không thể bắt đầu

- [ ] **Logo công ty** — file PNG nền trong suốt (hoặc SVG)
- [ ] **Ảnh sản phẩm** — mỗi SKU tối thiểu 1 ảnh, đặt tên theo mã SKU
- [ ] **Danh sách serial sản phẩm** — file Excel hoặc CSV, mỗi dòng 1 serial
- [ ] **Danh sách sản phẩm đầy đủ** — tên, mã SKU, giá, bảo hành (nếu chưa điền ở Phần 4)
- [ ] **Ảnh banner trang chủ** — kích thước tối thiểu 1920×600 px

### Tài liệu bắt buộc — cần có trước khi ra mắt

- [ ] **Chính sách bảo hành** — file Word/PDF
- [ ] **Chính sách đổi trả** — file Word/PDF
- [ ] **Chính sách bảo mật thông tin** — file Word/PDF *(bắt buộc để đăng app lên store)*
- [ ] **Điều khoản sử dụng** — file Word/PDF
- [ ] **Nội dung trang Giới thiệu** — file Word hoặc ghi chú

### Tài liệu kỹ thuật — đội lập trình sẽ hỗ trợ tạo hoặc hướng dẫn

- [ ] **File Firebase** (`google-services.json` cho Android, `GoogleService-Info.plist` cho iOS) — *cần tạo tài khoản Firebase trước*
- [ ] **Tài khoản Google Play Developer** — *nếu muốn đăng app Android*
- [ ] **Tài khoản Apple Developer** — *nếu muốn đăng app iPhone*

### Tài liệu nên có thêm

- [ ] **Bài viết blog / tin tức** — file Word hoặc PDF, mỗi bài 1 file
- [ ] **Video giới thiệu công ty hoặc sản phẩm** — link YouTube hoặc file MP4
- [ ] **Ảnh đội ngũ / văn phòng / kho hàng** — dùng cho trang Giới thiệu
- [ ] **Danh sách đại lý hiện có** — file Excel (tên, email, SĐT, hạn mức công nợ)
